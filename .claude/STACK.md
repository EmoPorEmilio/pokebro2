# Stack

**Bun** | **TanStack Start** | **SolidJS** | **Tailwind v4** | **Vite**

## Commands
```bash
bun install   # install
bun run dev   # dev server :3000
bun run build # production build
```

## Structure
- `src/routes/` - file-based routing
- `src/components/` - Solid components
- `src/styles.css` - Tailwind config (@theme)
